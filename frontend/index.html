How to Use (无障碍使用指南)

1. 启动系统

系统启动后会自动进入监听状态。您可以通过麦克风说出语音命令。

2. 指定目标物体

说 "Find bottle" 或 "Find cup" 来指定要抓取的物体。

3. 跟随语音指导

系统会不断播报方向指令（"Left", "Right", "Forward" 等）。 距离越近，播报频率越快，语速越快（类似倒车雷达）。

4. 完成抓取

当系统说 "Stop. Grasp now" 时，您可以伸手抓取物体。
